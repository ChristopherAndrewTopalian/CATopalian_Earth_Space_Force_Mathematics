# **JS**  
```javascript
// equation: 4x + 5 = 16

let total = 16;
let added = 5;
let coefficient = 4;

// subtract the added value from both sides
let reduced = total - added;

// divide by the coefficient of x
let x = reduced / coefficient;

console.log("The value of x is:", x);

// The value of x is: 2.75
```
